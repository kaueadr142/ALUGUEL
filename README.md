# Sistema de Aluguel de Equipamentos de TI

Projeto simples (HTML/CSS/JavaScript) para cadastro e controle de aluguel de equipamentos de tecnologia.
Dados são salvos no Local Storage do navegador.

Arquivos:
- index.html — interface principal
- styles.css — estilos
- app.js — lógica (persistência em localStorage, CRUD de equipamentos e aluguéis)
- README.md — este arquivo

Como usar:
1. Abra `index.html` em um navegador moderno.
2. Cadastre equipamentos no formulário.
3. Registre aluguéis, selecione equipamento, datas e locatário.
4. Exporte/importe dados em JSON para backup.

Observações:
- Sistema intencionalmente simples para uso local; não possui backend nem autenticação.
- Melhorias possíveis: validação mais robusta, relatório/filtragem, backend com banco de dados.
